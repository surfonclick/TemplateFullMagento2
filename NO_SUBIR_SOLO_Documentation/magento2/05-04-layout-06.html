Documentation

FashionNova Store

To make the theme look like this style:

  1. Active EM0144 FashionNova Store. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM Everything Homepage One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

Area01 - Main Slideshow

Area 02 - Banners

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Main Content Top - Four Banners (Identifier em0144-fashionnova-main-content-top-four-banners).

New Arrivals - Grid 21

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Main Content Area - New Arrivals - Grid 21 (Identifier em0144-fashionnova-main-content-area-new-arrivals-grid21).

In content of block, you double click on widget to configure widget options

Please go to EM Filter Products Widget to learn more about how to manage this widget.

Three Banners

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Main Content Area - Three Banners (Identifier em0144-fashionnova-main-content-area-three-banners).

From Our Blogs

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Main Content Area - Our Blogs (Identifier em0144-fashionnova-main-content-top-our-blogs).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Main Content Area - Our Blogs - Code (Identifier em0144-fashionnova-main-content-top-our-blogs-code). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - FashionNova Store - Main Content Area - Our Blogs and choose it.

Two Banners

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - FashionNova Store - Before Page Footer Container - Two Banners (Identifier em0144-fashionnova-before-page-footer-two-banners).
