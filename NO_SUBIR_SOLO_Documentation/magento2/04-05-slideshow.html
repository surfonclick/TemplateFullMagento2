EM Slideshow

Create a responsive (mobile friendly) or fullwidth slider with must-see-effects and meanwhile keep or build your SEO optimization (all content is always readable for search engines). See the heaps of custom transitions/animations for each object on the page! Customize this slider with our convenient drag & drop in the backend to your needs. This extensions features tons of unique transition effects, an image preloader, video embedding, autoplay that stops user interaction and easily set options to create your own effects.

1. Creating Slideshow

Back to top

1. In admin panel, select EMThemes > Manage Slideshows

2. Clicking on button to create a new slideshow.

Fill in the information of slideshow at General Setting tab.

Adding images to slideshow

Select image and style slideshow:

Besides, you can configure other parameters of a slideshow such as:

  • General tab:
    • General:
      • Name: Name of slideshow
      • ID: Identifier of slideshow. It must be unique.
      • Enable: It defines visibility of this block in the frontend.
    • Slider Size:
      • Slider Width: Width of slideshow.
      • Slider Height: Height of slideshow.
  • Effect Setting tab:

    You can customize the effect of the Slider in Slide Animation tab.
