Documentation

Bicycle

To make the theme look like this style:

  1. Active EM0144 Bicycle. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

EM0144 Everything - Bicycle - Area Navigation - Top Menu

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Main Megamenu (Identifier em0144-fashion-main-megamenu).

In content of block, you double click on Show/Hide Editor button to edit content as HTML tags

To show this menu on Clothing layout, we use a widget instance in Content > Widgets > EM0144 Everything - Bicycle Store - Main Megamenu

2. Main Content

Back to top

EM0144 Everything - Bicycle - Area01 - Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - Main Content Top - Main Slideshow (Identifier em0144-bicycle-main-content-top-main-slideshow).

In content of block, you double click on widget to choose Slideshow

To manage EM Slideshow, go to EM Slideshow Widget to learn more about how to edit content of a slideshow.

EM0144 Everything - Bicycle - Area 02 - Tabs Products

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle - Main Content Area - Tabs Products (Identifier em0144-bicycle-main-content-area-tabs-products).

EM0144 Everything - Bicycle - Area Main - Best Sellers

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - Main Content Area 02 - Best Sellers (Identifier em044-bicycle-main-content-area-02-best-sellers).

EM0144 Everything - Bicycle - Area 03 - Ads

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - Main Content Container - Banners (Identifier em0144-bicycle-main-content-container-banners).

EM0144 Everything - Bicycle - Area 04 - Ads

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - Main Content Container - Testimonials (Identifier em0144-bicycle-main-content-container-testimonials).

EM0144 Everything - Bicycle - Area 05 - From Our Blog - Ads10

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - After Main Content 03- Our Blogs (Identifier em0144-bicycle-after-main-content-03-our-blogs).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - After Main Content 03- Our Blogs - Code (Identifier em0144-bicycle-after-main-content-03-our-blogs-code). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - Bicycle Store - After Main Content 03- Our Blogs and choose it.

EM0144 Everything - Bicycle - Area 06 - Brands Slider

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Bicycle Store - Page Footer Before - Brand Slider (Identifier em0144-bicycle-page-footer-before-brand-slider).
