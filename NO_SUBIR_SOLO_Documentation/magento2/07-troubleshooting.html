3. Third-party extensions

Back to top

This theme may not be compatible with some third-party extensions. It’s impossible for theme author to make the theme compatible out of the box with all existing extensions because there are thousands of available Magento extensions.

Generally all extensions can work with all themes but sometimes they have to be integrated manually. That’s because the extension and the theme override the same Magento template file that causes a conflict. In such cases you need to customize the theme and merge two conflicting template files into one template file.

This isn't something specific to our themes, this is general rule for all Magento themes and extensions.

Unfortunately third-party extensions are beyond the scope of our support, please contact the author of the extension if needed. Author of the extension should provide detailed instructions of how to integrate the extension with any custom theme.

To work with this theme, an extension should be installed in theme's directories, not in the “default” Magento theme directories. To be more specific: extension should be installed in the main directories of the default theme in the Everything design package.

4. Fix issue http://192.168.1.7/em0144_everything/pub/media/emthemes/themesettings/ background/blank.gif

Back to top

The Issue only have on Full-packagetheme. Because you used the our database. So it will be saved on database

We have 2 options to fix that issues

Option 1

Open admin (backend) -> go to Emthemes -> Everything Settings

Select what themes which you want to use

Select "Typography Settings"

Select the background patern and save it.

After that, you need go to System -> Cache Management -> clear all cache magento

clear cache browsers

Option 2

open ssh (root account) -> on root folder (ex: var/www/html) run the command:

(example find windowns and replace it with linux )


grep -rl 'windows' ./ | xargs sed -i 's/windows/linux/g'

example1: our host: http://magento2.emthemes.com and link site is: everything. (url: http://magento2.emthemes.com/everything/)

the command

  • 1: sudo grep -rl '192.168.1.7' app/design/frontend/Emthemes/everything/* | xargs sed -i 's/192.168.1.7/magento2.emthemes.com/g'
  • 2: sudo grep -rl 'em0144_everything' app/design/frontend/Emthemes/everything/* | xargs sed -i 's/em0144_everything/everything/g'
  • 3: sudo php -f bin/magento setup:upgrade
  • 4: sudo php -f bin/magento setup:static-content:deploy
  • 5: sudo chmod -R 777 var/ pub/
  • 6: clear cache browsers

example2: your host: http://demo.magento2.com/

the command

  • 1: sudo grep -rl '192.168.1.7' app/design/frontend/Emthemes/everything/* | xargs sed -i 's/192.168.1.7/demo.magento2.com/g'
  • 2: sudo grep -rl 'em0144_everything' app/design/frontend/Emthemes/everything/* | xargs sed -i 's/em0144_everything///g'
  • 3: sudo php -f bin/magento setup:upgrade
  • 4: sudo php -f bin/magento setup:static-content:deploy
  • 5: sudo chmod -R 777 var/ pub/
  • 6: clear cache browsers

Done!
