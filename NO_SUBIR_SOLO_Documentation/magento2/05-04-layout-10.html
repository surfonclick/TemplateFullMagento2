Documentation

Baby

To make the theme look like this style:

  1. Active EM0144 Baby. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

1. Header

Back to top

Header Phone

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Area Header 01 - Phone (Identifier em0144-baby-area-header-01-phone).

Horizontal Menu

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Main Megamenu (Identifier em0144-fashion-main-megamenu).

In content of block, you double click on Show/Hide Editor button to edit content as HTML tags

To show this menu on Baby layout, we use a widget instance in Content > Widgets > EM0144 Everything - Baby Store - Main Megamenu

2. Main Content

Back to top

Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Area - Slideshow (Identifier em0144-baby-main-content-area-slideshow).

In content of block, you double click on widget to choose Slideshow

To manage EM Slideshow, go to EM Slideshow Widget to learn more about how to edit content of a slideshow.

Three Banners Below Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Area - Three Banners (Identifier em0144-baby-main-content-area-three-banners).

New Arrivals Slider

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Area - Home - New Arrivals Slider (Identifier em0144-baby-main-content-area-home-new-arrivals-slider).

In content of block, you double click on widget to configure EM Filter Products Widget

Please go to EM Filter Products Widget to learn more about how to manage this widget.

Best Sales

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Area - Home - Best Sale Slider (Identifier em0144-baby-main-content-area-home-best-sale-slider).

In content of block, you double click on widget to configure EM Filter Products Widget

Please go to EM Filter Products Widget to learn more about how to manage this widget.

Fullwidth Banner

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Area - Fullwidth Banner (Identifier em0144-baby-main-content-area-full-banner).

Featured Category

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Bottom - Featured Category (Identifier em0144-baby-main-content-bottom-featured-category).

Advertisement

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Bottom - Testimonial - Blog (Identifier em0144-baby-main-content-bottom-testimonial-blog).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Main Content Bottom - Testimonial - Blog - Code (Identifier em0144-baby-main-content-bottom-testimonial-blog-code). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - Baby Store - Main Content Bottom - Testimonial - Blog and choose it.

Newsletter

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Baby Store - Footer Area 01 - Newsletter (Identifier em0144-baby-footer-area-01-newsletter).
