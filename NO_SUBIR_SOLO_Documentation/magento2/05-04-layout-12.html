Documentation

Beauty Store

To make the theme look like this style:

  1. Active EM0144 Beauty. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Beauty layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Beauty - Main Slideshow (Identifier em0144-beauty-main-slideshow).

Main Menu

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Main Megamenu (Identifier em0144-fashion-main-megamenu).

In content of block, you double click on Show/Hide Editor button to edit content as HTML tags

To show this menu on Beauty layout, we use a widget instance in Content > Widgets > EM0144 Everything - Beauty Store - Main Megamenu

EM0144 Everything - Beauty - Main Content Area 05 - Blog - Testimonials

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Beauty - Main Content Area 05 - Blog - Testimonials (Identifier em0144-beauty-main-content-area-05-blog-testimonials).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Beauty - Main Content Area 05 - Blog - Code - Testimonials (Identifier em0144-beauty-main-content-area-05-blog-code-testimonials). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - Beauty - Main Content Area 05 - Blog - Testimonials and choose it.
