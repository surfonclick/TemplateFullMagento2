Problems after installing theme

How To Install Magento 2 and EM Everything on EC2 Amazon Server

  • 1. No style and layout available in the website
  • 2. Unable to access settings of theme and extension
You can read Troubleshoot from Magento about the installation errors and suggested solutions

1. No style and layout available in the website

Back to top

You need to flush Magento cache .

Go to System > Cache Management screen and use Flush cache button.

2. Can’t access settings of theme and extension

Back to top

Log out admin panel and log in again.

Or Save Administrator Roles again.
