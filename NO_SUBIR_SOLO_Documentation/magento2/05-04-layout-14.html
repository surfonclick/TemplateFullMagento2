Documentation

Supermarket Store

To make the theme look like this style:

  1. Active EM0144 Supermarket. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Supermarket layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

1. Header

Back to top

EM0144 Everything - Supermarket Store - Area Header 01 - Phone

In admin panel, select CONTENT > Blocks, look for block with name EM0144 Everything - Supermarket Store - Area Header 01 - Phone (Identifier em0144-supermarket-area-header-01-phone).

Main Menu

In admin panel, select CONTENT > Blocks, look for block with name EM0144 Everything - Main Megamenu (Identifier em0144-main-megamenu).

In content of block, you double click on widget to choose Menu

In content of block, you double click on Show/Hide Editor button to edit content as HTML tags

To show this menu on Health layout, we use a widget instance in Content > Widgets > EM0144 Everything - Supermarket - Main Menu
