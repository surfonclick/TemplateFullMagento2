1. Manage Category

Back to top

To manage categories, in admin panel go to Product > Categories.

If you install fresh Magento without sample data, the category tree only has a root category (Root Catalog)

The root category isn’t displayed in the frontend, and each store in Magento has only a root category, when you create or edit store (read more in section To create a store view), you will select this option.

You need some new categories to show in the frontend, and make sure that these categories are subcategory of root category

To add a new category, select the root category by clicking on it, then click on 'Add Subcategory' button

Fill in all the required fields

If the category is not displayed in the frontend: you need to reindex the data and flush Magento cache

2. Include category on menu

Back to top

You can include or exclude a category in Navigation Menu, go to Catalog > Manage Categories, select a category, in General Information tab, select 'Yes' or 'No' for Include in Navigation Menu option.

3. Category page

Back to top

You can select to display mode of a category page, it can be a landing page.

Go to Product > Categories, select a category, in Display Settings tab, select appropriate value in the Display Mode field, if your option has 'Static Block', then in the CMS Block field, select the block you would like to display on category page. The list will include all existing static blocks (CMS > Static Blocks)

4. Layered Navigation (Shop By block)

Back to top

Layered Navigation helps your customer have the option to filter through interests regardingless its price, manufacturer or any other aspect of products, and you can show customers what they want and raise conversions

To add Layered Navigation in a category page, go to Catalog > Manage Categories, select category, and set 'Is Anchor' field (in Display Settings tab) to Yes

To add more attributes to be displayed in Layered Navigation: go to Stores > Attributes > Product > [select attribute], Storefront Properties tab, set value 'Use in Layered Navigation' field to Yes. Please refer Magento documentation for more details:

http://www.magentocommerce.com/wiki/welcome_to_the_magento_user_s_guide/chapter_3#attributes
