Documentation

Swatch Store

To make the theme look like this style:

  1. Active EM0144 Swatch Store. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

EM0144 Everything - Swatch Store - Area Navigation - Top Menu
