Documentation

Fashion

To make the theme look like this style:

  1. Active EM0144 Magely Fashion. Go to Active Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM Everything Homepage Two Columns Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Clothing layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

2. Main Content

Back to top

Advertisement above Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Home - Ads Text (Identifier em0144-fashion-home-ads-text).

Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Area Main - Slideshow (Identifier em0144-fashion-main-area-main-slideshow).

In content of block, you double click on widget to choose Slideshow

To manage EM Slideshow, go to EM Slideshow Widget to learn more about how to edit content of a slideshow.

Three Banners Below Main Slideshow

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Area Main 01 - Home - Banners (Identifier em0144-fashion-area-main01-home-banners).

New Arrivals Tabs

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Area Main 02 - Home - Tabs (Identifier em0144-fashion-area-main02-home-tabs).

In content of block, you can edit Title of each tab

Content of each tab is EM Filter Product widget, you can double click on these widgets to edit widget options

If you would like to add more tabs, you need to click on Show/Hide Editorbutton to edit content as HTML tags and add tab as built-in structure

Best sales

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Area Main 04 - Home - Best Sale (Identifier em0144-fashion-area-main02-home-best-sale).

Please go to EM Filter Products Widget to learn more about how to manage this widget.

Sidebar Blog

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Sidebar Left 03 - Home - Recent Blog (Identifier em0144-fashion-sidebar-left03-home-recent-blog).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Sidebar Left 03 - Home - Recent Blog - Code (Identifier em0144-fashion-sidebar-left03-home-recent-blog-code). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - Fashion Store - Sidebar Left 03 - Home - Recent Blog and choose it.
