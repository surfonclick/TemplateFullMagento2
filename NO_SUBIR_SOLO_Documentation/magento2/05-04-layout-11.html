Documentation

Lingerie Store

To make the theme look like this style:

  1. Active EM0144 Lingerie. Go to Active & Deactive Theme to view demo of how to active & deactive.
  2. Edit Homepage to EM0144 Everything - Homepage - One Column Go to Edit CMS Home Page to view demo of how to set homepage.

To show blocks below, we created widget instance (Content > Widget) with same name of blocks, you can edit its position in widget instance by modifying current layout or add more layouts.
Some blocks of Lingerie layout use common static blocks of default layout (Fashion Layout), please click on content of each block to view details.

EM0144 Everything - Lingerie Store - Area Navigation - Main Menu

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Fashion Store - Main Megamenu (Identifier em0144-fashion-main-megamenu).

EM0144 Everything - Lingerie Store - Main Content Bottom - From Our Blog

In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Lingerie Store - Main Content Bottom - From Our Blog (Identifier em0144-lingerie-main-content-bottom-blog).

In our theme, we are using Magefan Blog extension to demo Blog page. You can visit this link view more details about it: http://magefan.com/magento2-blog-extension/
In case you installed Magefan Blog or would like to use this extension, you can contact us, we will provide you the templates and css styles of blogs on EM Everything theme.

We also created a Static Block which use Magefan Blog to show recent blogs on sidebar. You can find it In admin panel, select Content > Blocks, look for block with name EM0144 Everything - Lingerie Store - Main Content Bottom - From Our Blog - Code (Identifier em0144-lingerie-main-content-bottom-blog-code). To replace Recent Blogs by this block, you go to Content > Widgets > edit EM0144 Everything - Lingerie Store - Main Content Bottom - From Our Blog and choose it.
